# Vue

## 3.3 → 3.4 (2023-12-28)

**Changes**

- Rewrote the template parser, with 2x faster parser and improved SFC build performance.
- Refactor the reactivity system, making it more efficient.
- `defineModel` is now stable.
- `v-bind` same-name shorthand.
- Improved hydration mismatch errors.
- Error code and compile-time flag reference.
- Removed global JSX namespace.
- Removed reactivity transform.
- Removed `app.config.unwrapInjectedRef`.
- Removed `v-is` directive.
- Removed `@vnodexxx` event listeners.
- MathML support.

**Code update**

Upgrade packages: `Volar/vue-tsc@^1.8.27, @vitejs/plugin-vue@^5.0.0, nuxt@^3.9.0, vue-loader@^17.4.0`.

Update `tsconfig.json` if using tsx.

```json
{
  "jsxImportSource": "vue"
}
```

```html
<img :id="id" :src="src">
↓↓↓
<img :id :src>
```

```html
<div v-is="dynamic-component-name" />
↓↓↓
<div is="vue:dynamic-component-name" />
```

## 3.2 → 3.3 (2023-05-11)

**Changes**

- Imported and complex types support in macros.
- Generic components.
- More ergonomic `defineEmits`.
- Typed slots with `defineSlots`.
- `defineModel`.
- `defineOptions`.
- Reactive props destructure.
- Add `jsxImportSource` in `tsconfig.json`.

**Code update**

```ts
interface Props {}

defineProps<Props>()

//↓↓↓

import type { Props } from './foo'

defineProps<Props>()
```

```ts
const emit = defineEmits<{
  (e: 'foo', id: number): void
  (e: 'bar', name: string, ...rest: any[]): void
}>()

//↓↓↓

const emit = defineEmits<{
  foo: [id: number]
  bar: [name: string, ...rest: any[]]
}>()
```
