# [Helmet](https://github.com/helmetjs/helmet/blob/main/CHANGELOG.md)

## 6.0 → 7.0 (2023-05-06)

**Changes**

- `Cross-Origin-Embedder-Policy` middleware is disabled by default.
- Remove `Expect-CT`, use `expect-ct` package instead.
- Drop support for Node 14 and 15. Node 16+ is now required.

---

## 5.0 → 6.0 (2022-08-26)

**Changes**

- `helmet.contentSecurityPolicy` no longer sets `block-all-mixed-content` directive by default.
- `helmet.expectCt` is no longer set by default. It can, however, be explicitly enabled.
- `helmet.frameguard` no longer offers a specific error when trying to use `ALLOW-FROM`; it just says that it is
unsupported. Only the error message has changed.
- Dropped support for Node 12 and 13. Node 14+ is now required.

---

## 4.0 → 5.0 (2022-01-02)

**Changes**

- Added ECMAScript module imports
- `helmet.contentSecurityPolicy`: `useDefaults` option now defaults to `true`
- `helmet.contentSecurityPolicy`: `form-action` directive is now set to `self` by default
- `helmet.crossOriginEmbedderPolicy`, `helmet.crossOriginOpenerPolicy`, `helmet.crossOriginResourcePolicy`, `helmet.originAgentCluster` are enabled by default
- Drop support for Node 10 and 11. Node 12+ is now required
