# Vue-i18n

## 10.0 -> 11.0 (24-12-24)

**[Changes](https://github.com/intlify/vue-i18n/releases/tag/v11.0.0)**

- Deprecated Legacy API mode.
- Deprecated `v-t`.
- Drop `tc` and `$tc` for Legacy API mode.

## 9.0 -> 10.0 (24-09-10)

**[Changes](https://github.com/intlify/vue-i18n/releases/tag/v10.0.0)**

- A minimal `petite-vue-i18n`.
- JIT Compilation Enabled by Default.
- Support for Generated Locale Types.
- Deprecation of `tc` and `$tc`.
